# Getting Started

## Setup {#setup}
> __Most of documentation pertaining to general configuration for `sqler-mdb` can be found in the [`sqler` manual](https://ugate.github.io/sqler).__

Install the following modules when using `sqler-mdb` for [MariaDB](https://mariadb.org) and/or [MySQL](https://www.mysql.com):
```sh
npm install sqler sqler-mdb mariadb
```

Connection and execution option extensions can be found in the [API docs](/api/).
